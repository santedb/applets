Welcome To SanteDB!

Welcome to SanteDB Version {{$root.system.version}}. This is the default applet welcome screen. You will need to install one or more plugins into the SanteDB server in order for more useful screens to appear.

Configuration might be needed

You may also be seeing this page because you have not fully configured this instance of the SanteDB disconnected service you are running. To configure this instance jump over to the Initial Configuration Screen

Developing your own app?

If you're developing your own application, you have configured your instance and still see this page, try adding a reference to your application source code directory by using:

sdb-are --ref=org.santedb.core.pak --ref=org.santedb.uicore.pak --applet=path_to_your_applet

However, if you're seeing this message on a mobile device, a disconnected server, or production (non-development tool) it may be your application is not registered or does not have a startup asset.

Please contact your system administrator for more information.
